# walmart-sales-time-series
End-to-end time series forecasting pipeline for Walmart sales. Features exhaustive EDA, seasonality decomposition, and a 32.2% MAPE reduction using SARIMA, SARIMAX, and Prophet. Validated the impact of holidays and macro-economics on retail demand using 143 weeks of data across 45 stores. Optimized via grid search for high-accuracy demand planning.
